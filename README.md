# «Альтернативные источники данных о смертях публичных фигур»

Хотя Википедия остаётся одним из самых удобных и структурированных источников информации о смертях известных личностей, <br>
полагаться только на неё — рискованно. Обновление страницы «Deaths in…» может задерживаться на часы или даже дни, <br>
особенно в регионах с низкой плотностью вики-редакторов. <br>
Кроме того, страница охватывает лишь наиболее «заметных» персон, <br>
оставляя за рамками многих деятелей культуры, науки или региональной политики.<br>
<br>
Существует ряд альтернативных технических подходов, позволяющих отслеживать подобные события быстрее и шире:<br>
<br>
## Новостные агрегаторы и RSS-каналы

Сервисы вроде Google News, Associated Press, Reuters или BBC часто публикуют некрологи в течение нескольких часов после события. <br>
С помощью RSS-парсеров (например, feedparser) и NLP-моделей можно автоматически извлекать упоминания смертей. <br>
Ключевые фразы: “dies at”, “passed away”, “death announced”.<br>

## Twitter / X и социальные сети

Смерть известной личности почти мгновенно распространяется через Twitter. <br>
Используя Twitter API (или бесплатные альтернативы вроде snscrape), <br>
можно отслеживать треды с высокой виральностью или официальные аккаунты СМИ. <br>
Однако требуется фильтрация дезинформации.<br>

## Специализированные базы данных и API  

Obituary services: Legacy.com, Find a Grave, BillionGraves — содержат структурированные данные, но доступ через API ограничен.  <br>
Wikidata: более машинно-ориентированная альтернатива Википедии. <br>
OpenCorporates / Who’s Who: иногда содержат даты смерти для государственных и деловых деятелей.<br>

## Web-скрапинг региональных СМИ

Для локальных деятелей (артистов, учёных, политиков) стоит мониторить национальные газеты и агентства (например, TASS, Le Monde, El País). <br>
Автоматизация через BeautifulSoup + регулярные выражения или Playwright для динамических сайтов.<br>

## Telegram-каналы и чаты

Многие профессиональные сообщества (журналисты, историки, IT-эксперты) оперативно делятся новостями в Telegram. <br>
Можно использовать Telegram Bot API для мониторинга таких каналов.<br>
